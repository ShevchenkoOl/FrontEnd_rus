Числа и строки в JavaScript
1. Числа в JavaScript
    JavaScript, в отличии от других языков програмирования имеет всего один тип данных — Number. Он универсальный и подходит для:
                        - целых чисел (10, -5)
                        - дробных чисел (3.14)
                        - специальных значений (Infinity, NaN)
    Любое число в JavaScript с помощью стандарта IEEE 754 автоматически преобразуется к единому 64-битному формату.
    IEEE 754 — это международный стандарт, который определяет, как должно храниться число в памяти компьютера.

    Что такое 64-бита - это фиксированное количество ячеек памяти, которое JavaScript выделяет, чтобы записать одно число. Компьютер хранит данные в виде битов:
                                            - 1 бит — это либо 0, либо 1
                                            - 8 бит = 1 байт
                                            - 64 бита = 8 байт
    Важно чётко понимать что 64-бита, это не количество цифр в числе, а количество ячеек. Потомучто не только цыфры имееют размер 1 бит, но и знаки (+ или -), степени (эксрлнент) - 11 бит и мантиссы(точность числа) - 52 бита.
    Не зависимо от длинны числа всё равно обьём будет 64-бита.
        Например:
        - число 5
        - число 1000000000000
        - число 3.1415926535
        - число 99999999999999999999999999
        - даже Infinity


1.1 Преобразование строки в число
Часто данные от пользователя приходят как строки (например, через prompt()).
Пример:
const age = prompt("Введите возраст");
console.log(age); // например "25" → строка

    Чтобы работать с этим как с числом — нужно перевести строку в число.
        Способы перевода строки в число:
            - Number() — универсальный способ
                let n = Number("25"); // 25
                let x = Number("3.14"); // 3.14
                let bad = Number("abc"); // NaN
            - parseInt() — переводит в целое число
                parseInt("42")          // 42
                parseInt("42px")        // 42
                parseInt("3.99")        // 3
            - Работает до первого «нечислового» символа. parseFloat() — переводит в дробное число
                parseFloat("3.14")      // 3.14
                parseFloat("5.7cm")     // 5.7
            - Унарный плюс, это оператор + , который применяется к одному значению, чтобы преобразовать его в число. Он называется унарным, потому что работает с одним операндом (в отличие от обычного +, который соединяет два значения — это бинарный оператор, то есть работает с двумя операндами а + с).
            Унарный плюс считаеться самый короткий и распространённый способ для перевода текста в число:
                    let a = +"50";  // 50
                    let b = +"3.14"; // 3.14
                    const value = prompt("Введите число:");
                    const number = +value;
                    let c = +"abc";  // NaN
            ⚠️ Важно помнить
            Если строка не может быть интерпретирована как число — результат будет NaN.

1.2 дробные числа (числа с плавающей точкой)
    JS хранит числа в двоичной форме ⇒ не все дроби можно представить точно.
    Пример известной «ошибки»:
        console.log(0.1 + 0.2); // 0.30000000000000004
        Причина: ограничения бинарной арифметики, то есть 64-бита про которые мы говорили выше не хватает для переоброзование дробей. Например:
                                        1/3 = 0,3333333333333333 - бесконечное 
                                        console.log(9999999999999999); // 10000000000000 - последние число просто округлилося
        Простыми словами говоря - это не ошибка JavaScript, а ограничение стандарта IEEE 754 и бинарной (двочной) арифметики, которая используется ВСЕМИ языками программирования.
        Ещё пару примеров: 
                        - console.log(0.3 - 0.1); // 0.19999999999999998
                        - Умножение "нечётных" дробей console.log(0.1 * 0.2); // 0.020000000000000004
                        - Вычитание console.log(0.3 - 0.1); // 0.19999999999999998

        Как решать или как с эти бороться?
            - Округлить до нужного количества знаков
                let sum = (0.1 + 0.2).toFixed(2); // "0.30"
            ⚠ Возвращает строку → нужно снова превращать в число: Number(sum)  // 0.3
            - Умножать на 10, выполнять операции, делить обратно: (0.1 * 10 + 0.2 * 10) / 10  // 0.3

1.3 Math — встроенный математический объект
JavaScript предоставляет специальный объект Math, содержащий десятки полезных функций.
Основные методы Math:
 - Math.round() — округление
 - Math.round(2.5); // 3
 - Math.round(2.4); // 2

 - Math.floor() — округление вниз
 - Math.floor(2.9); // 2

 - Math.ceil() — округление вверх
 - Math.ceil(2.1); // 3

 - Math.random() — случайное число от 0 до 1
 - Math.random(); // 0.1954857...
 - Math.floor(Math.random() * 10) + 1; - случайное число от 1 до 10
 - Math.floor(Math.random() * (max - min + 1)) + min - формула для получения рандомного числа в разных диапозонах, где max - это верхняя граница, min - нижняя

 - Math.max() / Math.min()
 - Math.max(10, 5, 20); // 20
 - Math.min(10, 5, 20); // 5

 - Math.pow() — степень
 - Math.pow(2, 3); // 8 Современная замена — оператор **.

1.4 NaN — Not a Number
    NaN означает, что операция не может выдать корректное числовое значение.
        Примеры:
            Number("abc");      // NaN
            "hello" * 5;        // NaN
            undefined + 10;     // NaN
Особенности NaN:
❗ NaN не равен НИЧЕМУ, даже самому себе:
console.log(NaN === NaN); // false

✔ Проверка правильным способом:
Number.isNaN(value)
let value = "aas";
console.log(Number.isNaN(value)); // false - потому, что это проверка проверяет на число

1.5 undefined — это специальное значение в JavaScript, которое означает:
    «Переменная существует, но значение ей не присвоено»
    let a;
    console.log(a); // undefined
    Важно: undefined ≠ null

2. Строки в JavaScript
    Строка — индексованный и последовательный набор символов начиная с нуля, который записываеться в одинарные, двойные или обратные кавычки:
        "Hello"
        'Привет'
        `Шаблонная строка`
    Почему индексованный, потосучто кажлый символ строки имеет свой конкретный индекс:
        JavaScript
        0123456789
    Таким образом мы можем получить символа по индексу. Нумерация — с нуля:
        console.log("JavaScript"[2]); // v
        let str = "Hello";
        str[0]; // "H"
        str[4]; // "o"
    Важно: Строки неизменяемы → символы нельзя менять:
        let s = "Hi";
        s[0] = "h"; // не работает!

2.1 Свойства строк
Длина строки
"Hello".length;  // 5

2.2 Основные методы строк
    - toUpperCase / toLowerCase
       "alex".toUpperCase(); // "ALEX"
       "HELLO".toLowerCase(); // "hello"

    - indexOf — поиск подстроки
        "Hello world".indexOf("world"); // 6
        "Hello".indexOf("abc"); // -1

    - includes — проверка наличия
        "JavaScript".includes("Script"); // true

    - trim — удаляет пробелы по краям
        "   Alex   ".trim(); // "Alex"

    - slice — извлечение части строки
        "JavaScript".slice(0, 4); // "Java"
        "JavaScript".slice(4);    // "Script"

    - replace
        "Hello world".replace("world", "JS"); // "Hello JS"

    - repeat
        "Ha".repeat(3); // "HaHaHa"

    - Шаблонные строки и интерполяция. Используются обратные кавычки:
        let name = "Alex";
        console.log(`Привет, ${name}!`);
