1. Тег <button>
Тег <button> используется для создания интерактивных кнопок на веб-странице. Кнопки могут выполнять различные действия, например:
 - отправку формы на сервер,
 - выполнение кода JavaScript.

Пример кнопки:
<button type="button">Нажми меня</button>


 <button type="button" onclick="alert('Нажата стандартная кнопка')">Стандартная</button>
 
alert — это встроенная функция в JavaScript, которая используется для отображения всплывающего окна с сообщением пользователю.

Стандартная кнопка ничего не отправляет и используется с JS (onclick).

<!-- Кнопка отправки формы -->
<button type="submit">Отправить</button>
Submit отправляет форму на сервер.

<!-- Кнопка сброса формы -->
<button type="reset">Сбросить</button>
Reset возвращает все поля формы к исходным значениям.
    
<!-- Отключённая кнопка -->
<button type="button" disabled>Неактивная</button>
Disabled нельзя нажать, она визуально серого цвета.

Более подробнее разбёром как это работает при курсе на JS
Как можно стилизовать кнопку:

Использовать псевдоклассы (:hover, :focus, :active)

Задавать свойство cursor, чтобы изменить вид курсора при наведении

Псевдоклассы:

:hover – срабатывает, когда указатель мыши находится над элементом. Часто используется для изменения стиля кнопки при наведении.

button:hover {
    background-color: lightblue;
}

:focus – активируется, когда элемент в фокусе (например, выбран с помощью клавиши Tab).

input:focus {
    border-color: blue;
}

:active – срабатывает, когда элемент активен (например, когда нажата и удерживается кнопка мыши).


button:active {
    transform: scale(0.95);
}
Псевдоклассы можна использовать не тольео с кнопкам но и к другим тегам например картинка


Второе важное свойство при стилизации — это курсор

cursor — задаёт вид курсора мыши при наведении на элемент.

Существуют разные типы курсоров, например:
pointer — рука (обычно для кнопок и ссылок)
default — стандартный курсор стрелка
text — курсор ввода текста
wait — ожидание
crosshair — перекрестие
move — для перемещения

Box-model
Одна из сложных тем

Поэтому box-модель разделим на две части. Сегодня мы рассмотрим, что такое box для элементов, а на следующем уроке сосредоточимся на типах элементов и структурированных псевдоклассах.

Бокс-модель — это способ, которым браузер представляет любой HTML-элемент как набор прямоугольных «слоёв» (box). Каждый элемент состоит из четырёх частей (изнутри наружу):
        1. content — содержимое (текст, изображение).
        2. padding — внутренние отступы вокруг содержимого.
        3. border — рамка вокруг padding.
        4. margin — внешние отступы между этим элементом и соседями.

Эти свойства (ширина, высота, отступы и т.д.) можно задавать в разных единицах:
px – пиксели (фиксированный размер)
em – относительно размера шрифта родителя
rem – относительно размера шрифта корневого элемента (<html>)
% – процент относительно родительского контейнера

<div class="parent">
        <p>Rodič s šířkou 400px</p>
        <div class="box1">100px</div>
        <div class="box2">50%</div>
</div>

<div class="parent" style="font-size: 20px;">
        <p>Rodič s font-size: 20px</p>
        <div class="box3">2em</div>
        <div class="box4">2rem</div>
</div>

.parent {
            width: 400px; /* Rodič má pevnou šířku */
            border: 2px solid black;
            padding: 10px;
            margin-bottom: 20px;
        }
.box1 {
        width: 100px;  /* Pevná šířka v pixelech */
        height: 50px;
        background-color: lightblue;
        }
.box2 {
        width: 50%;    /* Zabírá 50 % šířky rodiče */
        height: 50px;
        background-color: lightgreen;
        }
.box3 {
        font-size: 20px; /* Nastavíme velikost písma */
        width: 2em;      /* 2× velikost písma = 40px */
        height: 50px;
            background-color: lightcoral;
        }
.box4 {
        width: 2rem;     /* 2× základní velikost písma (16px) = 32px */
        height: 50px;
        background-color: lightsalmon;
        }

Объяснение родителей:
Первый .parent (родительский контейнер)

width: 400px; – его ширина фиксирована.
.box1 { width: 100px; } – фиксированная ширина.
.box2 { width: 50%; } – займет 50% от ширины родителя (400px → 200px).
Второй .parent (с font-size: 20px;)

.box3 { width: 2em; } – 2× размер шрифта родителя (20px → 40px).
.box4 { width: 2rem; } – 2× базовый размер шрифта html { font-size: 16px; }, то есть 32px, НЕ зависит от родителя.
