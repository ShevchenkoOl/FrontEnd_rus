Заключительное занятие по HTML&CSS, и на последок, я для вас подготовил несколько интересных дополнительных вещей которые мы не разобрали на занятиях:
1. Прикольные библиотеки и инструменты
   - Анимации и эффекты:    
        Animate.css - готовые CSS-анимации (подключается через CDN)
        Hover.css - коллекция hover-эффектов https://ianlunn.github.io/Hover/
        AOS (Animate On Scroll) - анимации при прокрутке https://michalsnik.github.io/aos/
        Swiper - лучшая библиотека для каруселей
                стили для красивой картинки:
                        .swiper {
                            max-width: 1000px;  /* Максимальная ширина */
                            width: 100%;        /* На маленьких экранах - на всю ширину */
                            height: 600px;      /* Высота слайдера */
                            margin: 0 auto;     /* Центрирование */
                            border-radius: 15px; /* Скругленные углы */
                            overflow: hidden;    /* Скрываем выступающие части */
                            box-shadow: 0 10px 30px rgba(0, 0, 0, 0.3); /* Тень */
                        }

                        /* Изображения внутри слайдов */
                        .swiper-slide img {
                                width: 100%;
                                height: 100%;
                                object-fit: cover;  /* Обрезает изображение, сохраняя пропорции */
                                display: block;
                        }
        GLightbox для галерей - это самые современные и популярные варианты
