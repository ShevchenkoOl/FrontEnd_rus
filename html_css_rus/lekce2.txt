Разделение страницы на секции

В зависимости от содержимого основной части страницы её можно дополнительно разделять на секции.
Если у нас есть разный контент, например информация о услугах, команде, ценах или отзывах, мы можем использовать тег <section> для логического разделения этих блоков.

Правило:

На <section> делим только большой объём контента.

Меньший контент, как в шапке (<header>) или подвале (<footer>), можно разделить с помощью универсального тега <div>.

Списки (list) в HTML

Список — это элемент, который позволяет структурировать содержимое в удобной и читаемой форме.
Применяется для отображения связанных элементов, например: список покупок, шаги инструкции, меню на сайте.

Три основных типа списков:

Неупорядоченный список (<ul>)
Используется, когда порядок элементов не важен. Элементы отмечаются точкой или другим символом.

Пример: список ингредиентов:

<ul>
    <li>Мука</li>
    <li>Сахар</li>
    <li>Яйца</li>
</ul>


Упорядоченный список (<ol>)
Используется, когда порядок элементов важен. Элементы нумеруются.

Пример: шаги приготовления пирога:

<ol>
    <li>Разогрейте духовку до 180 °C.</li>
    <li>Смешайте ингредиенты.</li>
    <li>Вылейте тесто в форму и выпекайте 30 минут.</li>
</ol>


Список описаний (<dl>)
Используется для определений и пояснений. Содержит название (<dt>) и описание (<dd>).

Пример: словарь:

<dl>
    <dt>HTML</dt>
    <dd>Язык разметки для создания веб-страниц.</dd>
    <dt>CSS</dt>
    <dd>Каскадные таблицы стилей для оформления веб-страниц.</dd>
</dl>


Примечание:

В нашей работе мы будем использовать <ul>, так как это самый распространённый тег.

Правило: если есть два или более одинаковых элемента по смыслу — используем список.

Основы CSS

Структура CSS-правила:

селектор {
  свойство: значение;
}


Селектор — HTML-элемент или группа элементов, к которым применяется правило.

Свойство — CSS-свойство, которое мы хотим изменить (color, font-size, background-color).

Значение — новое значение свойства (red, 16px, #fff).

Как правильно использовать CSS:

Создаём файл с расширением .css, например styles.css.

В HTML подключаем его через <link> в <head>:

<head>
    <link rel="stylesheet" href="styles.css">
</head>


Селекторы и классы:

Селектор по тегу меняет стиль всех элементов с этим тегом (не всегда удобно).

Классы позволяют применить стиль к конкретным элементам. В HTML — через class, в CSS — через .:

.alert { font-size: 20px; font-weight: 500; }
.success { color: green; }
.warning { color: orange; }
.error { color: red; }

<p class="alert success">Account registration success</p>
<p class="alert warning">Attention, tariffs will be changed</p>
<p class="alert error">Transaction error</p>


Inline-стили (style внутри HTML) хороши только для небольших проектов. Для больших проектов используют внешние CSS-файлы, что облегчает поддержку и повторное использование стилей.

Цвета в CSS

RGB
Смешивание красного, зелёного и синего (Red, Green, Blue) в разных пропорциях позволяет получить любой цвет:

p {
  color: rgb(100%, 0%, 0%); /* ярко-красный */
}


HEX (шестнадцатеричный формат)
Цвет задаётся в шестнадцатеричной системе: цифры 0–9 и буквы A–F (или a–f).

RGBA
Расширение RGB с добавлением прозрачности (альфа-канал):

color: rgba(255, 0, 0, 0.5); /* красный с 50% прозрачностью */
